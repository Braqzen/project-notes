# Intro
This repository contains notes, learnings, take-aways from whatever material has been read. This could be from a book, open-source repository, a paper, or anything else.
The notes are taken in Obsidian which has its own semi-compatible Markdown renderer. That means if you're looking at these notes outside of Obsidian (such as through Github) they likely look strange (long paragrapghs, indentation issues) and have broken links.
